Most Recent Project - SiteLeaves

Now deployed on www.siteleaves.com

A web application where users are able to log in as admins and build their own website using in app tools. No tech skills are needed on the part of the user. Though originally implemented to allow small companies to maintain their own website over time without a software team, the codebase can be used to allow content creators easier access to sitebuilding.


  • Tech: ASP.NET Core API with Angular Frontend and MySQL database

  • Intended to be a 'one and done' sale to customers. Complete ownership and ability to maintain the site rests with them.

  • Built with modularity and scalability in mind.
